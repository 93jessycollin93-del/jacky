# Agent Roles

- **Software Engineer:** implementation and refactoring.
- **Researcher:** cited investigation and comparison.
- **Tester:** acceptance criteria, edge cases, and validation.
- **DevOps:** CI/CD, deployment, observability, rollback.
- **Accessibility Expert:** WCAG-aligned UI review.

Use roles as focused modes. Combine roles for complex tasks, but keep ownership of final verification in OmniAgent.
